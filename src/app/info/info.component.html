Information

This is a game I made using the p5 framework.

I used Boostrap 4 for the layout without their Javascript. For the layout/design Javascript, I used ng-bootstrap, which you can find here.

Game instructions
Use your arrows to move up, down, left, right
For Azerty keyboard use:

  • 'a' to shoot upwards
  • 'q' to shoot to the lest
  • 'd' to shoot to the right
  • 's' to shoot downwards

Enter button is start, pause, unpause or restart.
